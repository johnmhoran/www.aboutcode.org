## Ecma International approved ECMA-427 for PURL

*   **Date**: December 10.205
*   **Details**: The Package URL (PURL) standard has reached a major milestone
 with its approval as [the ECMA-427 standard](https://ecma-tc54.github.io/ECMA-427/).
 This validates PURL as the primary standard for software package 
 identification across the software supply chain. PURL is also on a fast-track
 for submission to [ISO/IEC JTC 1](https://www.iso.org/committee/45020.html)
 to become an ISO standard
